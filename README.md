# Secure Bank Network Design (Cisco Packet Tracer)

### Tools Used:
- Cisco Packet Tracer
- OSPF Routing
- DNS, DHCP
- Access Control Lists (ACLs)

### Features:
- Dynamic routing using OSPF
- Automatic IP Allocation using DHCP
- Secure access between departments using ACLs
- Hierarchical network design
